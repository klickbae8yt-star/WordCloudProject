Help

📁 File Upload: Choose an Excel file (.xlsx, .xls) or CSV file (.csv). Each sheet becomes a category.
📊 Categories: First column = words, second column = definitions, third column = category (optional).
🔍 Search: Type to filter words by name or definition.
🎨 Colors: Each category gets a unique color.
📥 Export: Download your cloud as a PNG image.
